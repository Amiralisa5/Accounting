# BigBang Cloud.ERP.Accounting

prerequisites
 - dotnet 8
 - nodejs 20+
 - powershell 7+
## Getting started
Install BigBang cli tools using below command
``` powershell
dotnet tool install bigbang.tool -g 
```
for prerelease version
```powershell
dotnet tool install bigbang.tool -g --prerelease
```
for build run
```powershell
./build.ps1
```
